# Version Collecte - GestionBornesCollecte

## C'est quoi cette version ?

Cette version utilise des données codées en dur, sans base de donnée.
C'est la premiere version que j'ai fait pour tester l'interface.

## Comment ca marche

Le service BorneService.cs retourne directement une liste de 5 bornes 
avec des niveaux de remplissage fixés.

## Pour lancer le projet

Rien a installer, ca marche directement dans Visual Studio.

## Pourquoi cette version

J'ai fait cette version pour pouvoir travailler et tester 
sans avoir besoin de la BDD de Diana.
C'est utile pour le développement et les tests.
